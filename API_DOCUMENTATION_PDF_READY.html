üîê Authentication Guide

Header Format:

Authorization: Bearer {access_token}

Token Lifetime: 24 hours

Token Format: Base64 encoded JSON dengan user info

Cara Penggunaan:

  1. Login menggunakan endpoint /api/login
  2. Simpan access_token dari response
  3. Gunakan token di header untuk endpoint yang memerlukan auth
  4. Refresh token jika expired (login ulang)
